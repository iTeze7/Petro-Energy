# Petro Energy

Site institucional corporativo completo em **Vue 3 + Vite** com layout premium, responsividade e acessibilidade.

## Estrutura
- `src/components/` — componentes reutilizáveis
- `src/pages/` — páginas do aplicativo
- `src/router/` — roteamento SPA com rotas dinâmicas
- `src/assets/` — ativos visuais (logo SVG)
- `src/styles/` — CSS global e variáveis
- `src/data/` — dados mockados para posts e indicadores

## Instalação
```bash
npm install
npm run dev
```

## Rotas
- `/` — Home
- `/sobre` — Sobre
- `/posts` — Lista de posts
- `/posts/:id` — Página de post dinâmica
- `/projetos-sociais` — Projetos sociais
